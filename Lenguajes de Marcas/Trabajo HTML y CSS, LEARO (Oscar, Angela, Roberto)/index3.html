<a>

La etiqueta <a> se utiliza para crear enlaces o hipervínculos a otras páginas web. El atributo href especifica la URL de destino.

Ejemplo 1: Visita nuestro sitio web aquí.

Ejemplo 2: Encuentra más información en la página de inicio.

<abbr>

La etiqueta <abbr> se usa para definir abreviaturas o acrónimos, y el atributo title proporciona la descripción completa de la abreviatura.

Ejemplo 1: WWW es la abreviatura de World Wide Web.

Ejemplo 2: El MIT es una prestigiosa universidad.

<b>

La etiqueta <b> se utiliza para aplicar formato de texto en negritas. Sin embargo, se recomienda utilizar <strong> para el significado semántico.

Ejemplo 1: Este texto está en negritas.

Ejemplo 2: Las instrucciones de seguridad son importantes.

<br>

La etiqueta <br> se emplea para insertar saltos de línea en un texto, lo que provoca un salto a la siguiente línea.

Ejemplo 1: Este es un ejemplo de línea 1.
Este es un ejemplo de línea 2.

Ejemplo 2: Use <br> para crear un salto de línea en HTML.

<cite>

La etiqueta <cite> se utiliza para marcar citas o referencias a obras como libros, películas, o fuentes confiables.

Ejemplo 1: La Enciclopedia Británica es una fuente confiable de información.

Ejemplo 2: El libro Matar a un ruiseñor es una obra clásica.

<code>

La etiqueta <code> se usa para mostrar texto en un tipo de letra monoespaciado, ideal para mostrar código fuente o comandos.

Ejemplo 1: Para imprimir "Hola, Mundo" en Python, use print("Hola, Mundo").

Ejemplo 2: El código <html> se utiliza para crear páginas web.

<del>

La etiqueta <del> se utiliza para marcar el texto que ha sido eliminado o tachado de un documento.

Ejemplo 1: El producto anterior ya no está disponible.

Ejemplo 2: El artículo 123 fue eliminado del catálogo.

<dfn>

La etiqueta <dfn> se emplea para definir términos o palabras clave y proporcionar su descripción.

Ejemplo 1: HTML es el lenguaje de marcado utilizado para crear páginas web.

Ejemplo 2: AT está disponible las 24 horas.

<em>

La etiqueta <em> se utiliza para dar énfasis o importancia a un texto dentro de un párrafo.

Ejemplo 1: La emoción se reflejó en su rostro.

Ejemplo 2: Es importante enfatizar la seguridad en el trabajo.

<i>

La etiqueta <i> se usa para aplicar formato de texto en cursiva a una sección de texto.

Ejemplo 1: Esto es cursiva.

Ejemplo 2: El término status quo se utiliza comúnmente.

<iframe>

La etiqueta <iframe> se utiliza para incrustar una página web o contenido externo dentro de una página web principal.

<ins>

La etiqueta <ins> se emplea para marcar el texto que ha sido insertado o añadido a un documento.

Ejemplo 1: El nuevo producto se lanzará pronto.

Ejemplo 2: La ley fue modificada el año pasado.

<kbd>

La etiqueta <kbd> se utiliza para mostrar entrada de teclado, como combinaciones de teclas o comandos.

Ejemplo 1: Para guardar un documento, presione Ctrl + S.

Ejemplo 2: Ingrese su nombre de usuario y contraseña para iniciar sesión.

<mark>

La etiqueta <mark> se utiliza para resaltar o destacar partes de un texto para mayor énfasis.

Ejemplo 1: Este texto está resaltado para mayor énfasis.

Ejemplo 2: El producto destacado de la semana es...

<q>

La etiqueta <q> se emplea para marcar citas en línea, como frases o fragmentos de texto citados.

Ejemplo 1: El autor dijo: La imaginación es más importante que el conocimiento.

Ejemplo 2: La cita famosa es: ¡Sí se puede!

<s>

La etiqueta <s> se utiliza para marcar texto tachado o eliminado de un documento.

Ejemplo 1: El precio original era $100, pero ahora está en $80.

Ejemplo 2: El modelo anterior ya no está en producción.

<samp>

La etiqueta <samp> se emplea para mostrar muestras de salida, como resultados de comandos o programas.

Ejemplo 1: El resultado del comando es: ls -l

Ejemplo 2: La salida del programa es: Error: archivo no encontrado

<small>

La etiqueta <small> se utiliza para aplicar un estilo de texto más pequeño.

Ejemplo 1: Este es un texto más pequeño en tamaño.

Ejemplo 2: Aviso legal: Términos y condiciones se aplican.

<span>

La etiqueta <span> se emplea para aplicar estilos o atributos a una sección específica de texto en línea.

Ejemplo 1: Este es un texto con estilo.

Ejemplo 2: El texto resaltado es importante.

<strong>

La etiqueta <strong> se utiliza para enfatizar o resaltar el texto, dándole un mayor peso semántico.

Ejemplo 1: Este es un texto fuertemente enfatizado.

Ejemplo 2: Utilice contraseñas seguras para proteger su cuenta.

<sub>

La etiqueta <sub> se emplea para marcar el texto en subíndice, como fórmulas químicas o matemáticas.

Ejemplo 1: H2O es la fórmula del agua.

Ejemplo 2: CO2 es dióxido de carbono.

<sup>

La etiqueta <sup> se utiliza para marcar el texto en superíndice, como exponentes matemáticos.

Ejemplo 1: 23 es igual a 8.

Ejemplo 2: x2 + y2 = r2

<time>

La etiqueta <time> se utiliza para representar fechas y horas específicas.

Ejemplo 1: La fecha de entrega es el 15 de noviembre de 2023.

Ejemplo 2: La reunión comienza a las 4:00 PM.

<var>

La etiqueta <var> se utiliza para representar variables en programación o matemáticas.

Ejemplo 1: Si x es igual a 5, entonces x + 3 es igual a 8.

Ejemplo 2: En programación, las variables se declaran usando la palabra clave var.
